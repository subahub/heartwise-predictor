# HeartWise – Heart Disease Risk Prediction System

## 📌 Overview

HeartWise is a full-stack web application designed to predict the risk of cardiovascular disease based on user-provided health parameters.

The system combines a modern frontend interface with a machine learning–based backend to deliver structured and instant heart risk analysis.

---

## 🚀 Features

- Interactive health parameter input form  
- Instant risk prediction results  
- Clean and responsive user interface  
- REST API integration  
- Modular and scalable architecture  

---
